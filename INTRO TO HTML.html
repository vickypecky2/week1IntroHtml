INTRODUCTION TO HTML

HTML- Hypertext Markup Language

Html is the foundation on which all other languages are built. It is everywhere round about us. HTML is the markup our content, the language that helps to communicate what humans means to the computer. An HTML tag is a special word or letter surrounded by angle brackets, < and >. Many elwmwnts have an opening tag and closing tag while some elements don't have a closing tag. These are called empty elements. e.g br element fr inserting line breaks is simply written < br >

THE ROLES OF HTML

  • It help defines the content of a site

HTML TAGS

  • <p> This is a Paragraph. </p>

  • <h1>This is a head tag </h1> The heading tag has grade from h1 to h6 i.e from highest to lowest

  • <article> This is an article. </article>

  • <ul> This is the tag for unorder list </ul>

  • <ol> This is the tag for order list </ol>

  • <li> This is the tag for listing items </li>

There are lots to learn in HTML, we have table tags, button tags, and syntax to creating forms.
